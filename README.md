# Whistle API
## Environment setup 

#### Requirements

• MongoDB

• NodeJs

## Run commands
```
```
#### Start mongo service
```
$ mongod
$ mongo
```
#### Open project directory
```
$ cd api_whistle/whistle

```
#### Do this only on the first run
```
$ npm install
```

#### Start node service
```
$ node .

$ Load url on web browser to access explorer

```
